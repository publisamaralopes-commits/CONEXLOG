LOTE STRUCTURE
==============

Each lote contains:

* Cliente
* Origem
* Cidade Origem
* Destino
* Cidade Destino
* Valor do Frete (optional)

STATUS SUMMARY:

* Veículos em Trânsito
* Na Porta
* Carregados

==================================================
UPDATE POSITION SYSTEM
======================

When the user updates a lote position:

* update vehicle quantities
* save timestamp
* save responsible user
* register operation history

* carregados
* updated by
